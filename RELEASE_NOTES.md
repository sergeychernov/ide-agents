# Release notes

## 0.8.0 (не опубликована)

**Router-агенты** — теперь агент может делегировать работу другим агентам, а не только навыкам.

Раньше агент во frontmatter указывал только `skills`. Этого не хватало «роутерам» — агентам, которые распределяют работу между подчинёнными агентами. Теперь у router-агентов есть поле `subagents` — список идентификаторов делегируемых агентов.

Что это даёт:

- **Включил router — установились все подчинённые.** ide-agents автоматически включает делегированных subagent'ов рекурсивно (с защитой от циклов), вместе с их навыками.
- **Нельзя сломать router.** Пока router установлен в области, его subagent'а в той же области нельзя выключить — переключатель заблокирован с пояснением.
- **Аккуратное выключение.** При выключении router предлагается заодно убрать его subagent'ов и навыки — только те, которыми больше никто не пользуется, с выбором галочками.
- **Видно в карточке.** У router-агентов появился блок `Subagents` с бейджами и описаниями.

`subagents` — только для router-агентов; обычные агенты по-прежнему указывают только `skills`.
